# Company Logo Images

Please add the following logo image files to this directory:

## Required Logo Files:

1. **fortis-logo.png** - Fortis logo
2. **unilever-logo.png** - Unilever logo (white "U" with icons on blue background)
3. **usha-logo.png** - USHA logo (white letters in red squares)
4. **glenmark-logo.png** - Glenmark logo (red capsule with "e" cutout)
5. **ongc-logo.png** - ONGC logo (Hindi script with oil rig graphic)
6. **tech-mahindra-logo.png** - Tech Mahindra logo (grey "Tech" and red "Mahindra")
7. **pnb-metlife-logo.png** - PNB MetLife logo (maroon emblem with yellow accent)
8. **shriram-general-insurance-logo.png** - SHRIRAM General Insurance logo (person illustration with yellow bar)
9. **bajaj-allianz-logo.png** - Bajaj Allianz logo (blue background with white text)
10. **abbott-logo.png** - Abbott logo (blue stylized "a" with black text)

## How to Add the Images:

### Option 1: Run the Download Script
1. Open PowerShell in the project root directory
2. Run: `.\public\images\download-logos.ps1`
3. The script will attempt to download all logos automatically

### Option 2: Manual Download
If the script doesn't work, you can manually download each logo:

1. **Fortis**: https://logos-world.net/wp-content/uploads/2020/11/Fortis-Logo.png
2. **Unilever**: https://logos-world.net/wp-content/uploads/2020/04/Unilever-Logo.png
3. **USHA**: https://logos-world.net/wp-content/uploads/2020/11/Usha-Logo.png
4. **Glenmark**: https://logos-world.net/wp-content/uploads/2021/08/Glenmark-Logo.png
5. **ONGC**: https://logos-world.net/wp-content/uploads/2021/08/ONGC-Logo.png
6. **Tech Mahindra**: https://logos-world.net/wp-content/uploads/2020/11/Tech-Mahindra-Logo.png
7. **PNB MetLife**: https://logos-world.net/wp-content/uploads/2021/08/PNB-MetLife-Logo.png
8. **SHRIRAM General Insurance**: https://logos-world.net/wp-content/uploads/2021/08/Shriram-General-Insurance-Logo.png
9. **Bajaj Allianz**: https://logos-world.net/wp-content/uploads/2021/08/Bajaj-Allianz-Logo.png
10. **Abbott**: https://logos-world.net/wp-content/uploads/2020/04/Abbott-Logo.png

Right-click each link, "Save As", and save to this `public/images/` folder with the exact filenames listed above.

### Option 3: Use Your Own Logo Files
If you have the logo files from the companies, simply copy them to this folder with the exact filenames listed above.

## Notes:
- All images should be in PNG format
- Recommended size: approximately 140x60 pixels or larger (will be scaled down)
- Images should have transparent backgrounds where appropriate
- The website will automatically display these logos once the files are added to this folder
